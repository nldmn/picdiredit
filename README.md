# picdiredit

Small Python starter project for simple image editing utilities.

## What's included

- `src/picdiredit` — package source
- `tests/` — basic pytest tests
- `pyproject.toml` — project metadata
- `requirements.txt` — runtime/dev dependencies
- `picdiredit.code-workspace` — VS Code workspace file

## Quickstart

Create a virtual environment and install dependencies:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Run the CLI help:

```bash
python -m picdiredit.cli --help
```
